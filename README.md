# XDM - a simple display manager

## Setup for VoidLinux

Conifg files in: 

```
/etc/X11/xdm/
├ Xaccess
├ Xresources
├ Xservers
└ xdm-config
```

```
/usr/lib/X11/xdm/
├ GiveConsole
├ TakeConsole
├ Xreset
├ Xsession
├ Xsetup_0
├ Xstartup
├ Xwilling
├ libXdmGreet.so
└ pixmaps/
  ├ xorg-bw.xpm
  └ xorg.xpm
```
